Flipboard is a small implementation of JQuery, HTML, Bootstrap and CSS concepts. This is a simple web page creted to implement some of the JQuery concepts like toggling the display. Click on the "More" button provided in the navigation bar, the result toggles between hide and show the drop down list. Also click on the arrow provided at the bottom of the images, left arrow will change the images by sliding them left to right and also chnges the focus of the dots provided in between two arrows. Similarly right arrow will move the images from rigt to the left and also the position of focusing dot.
